# Diagram Files

This directory contains architecture diagrams in multiple formats:

## Available Formats

1. **Mermaid Diagrams** (in ARCHITECTURE.md)
   - Best for GitHub/GitLab markdown rendering
   - View directly in the repository
   - No additional tools needed

2. **PlantUML Diagrams** (.puml files)
   - Professional UML diagrams
   - Can be rendered with PlantUML tools
   - Supports more complex diagrams

## How to View

### Mermaid Diagrams
- View directly in `ARCHITECTURE.md` on GitHub
- Or use online Mermaid editor: https://mermaid.live/

### PlantUML Diagrams
1. Install PlantUML: http://plantuml.com/starting
2. Use VS Code extension: "PlantUML"
3. Or use online editor: http://www.plantuml.com/plantuml/uml/

## Diagram Types

- **architecture.puml**: System architecture overview
- **sequence.puml**: Request/response sequence flow
